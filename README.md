# mapleaftlet

Mapa com três camadas, feito com Javascript. 

- Duas camadas de satélite e uma street.
- Marcadores de pontos definidos.
- Pesquisa completa por Pais, Estado, Cidade, CEP, Rua, Nome de referência, Latitude e Longitude.
